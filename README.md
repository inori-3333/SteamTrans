# Steam 账本

一款完全离线的 Android Steam 市场交易账本，用于记录皮肤、宝石袋、宝石、卡牌、卡包、背景和表情的持仓、转换与已实现盈亏。

## 账务口径

- 金额统一为人民币，数据库以分为单位保存。
- 买入成本包含手续费；卖出收入扣除手续费。
- 同一物品使用汇总移动成本，不追踪具体买入批次。
- 已实现盈亏 = 卖出净收入 - 对应卖出持仓的成本；未卖出的持仓不计盈亏。
- 总计收入是全部卖出交易扣除手续费后的实际到账累计。
- 持仓宝石单独统计；持仓物品数量不包含散装宝石。
- Steam 回本价 = 单件汇总成本 ÷ 85%，并向上取整到分。
- 转换不会产生利润，只把转出物品的汇总成本转移给产出物品。
- 删除 App 会同时删除本地数据。

## 构建

环境要求：Android SDK 35、JDK 17 或更高版本。

```powershell
.\gradlew.bat test assembleDebug
```

APK 输出：`app/build/outputs/apk/debug/app-debug.apk`。

## 使用顺序

1. 在“持仓”页新增物品。
2. 点击“记一笔”记录买入、卖出或转换。
3. 宝石袋和宝石转换时可使用快捷比例，也可以输入任意转换数量。
